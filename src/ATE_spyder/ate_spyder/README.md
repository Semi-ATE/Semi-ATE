# The semi-ate-spyder package

This package provides some plugin for the IDE [Spyder IDE](https://www.spyder-ide.org/). The plugin provides the possibility to create test projects in the context of semiconductor tests. The new added project type is called **Semi-ATE Project**. It will provide the test project structure. Further it allows to implement test programs by first defining some setups including: test-hardware, masksets, dies, packages etc.
